0) Create a new folder for your project
  git init
  git submodule add git@github.com:trevorwiebe/atomic-agents.git tools/atomic-agents

1) To update a project with the latest from this repo,
  git submodule init
  git submodule update

1) Create a virtual python environment
  python3 -m venv <myenvpath>
  source <myenvpath>/bin/activate

2) Install dependencies
  In the folder where setup.py is located, run pip install .
  Make sure you can see /atomic-agents/main/main.py
  Create a .env file in /atomic-agents/main/ and add the model api key

3) Start the script.
  python3 atomic-agents/main/main.py
